# 💻 Gatsby Bienes Raices

Gatsby Bienes Raices es un sitio estatico que recibe información desde Strapi

![Gatsby Bienes Raices](.readme-static/Home.png)

## 🛠 Scripts

```bash
# Instalar dependencias
$ npm install

# Encender servidor de desarrollo en localhost:8000
$ npm run develop

# Crear version de producción
$ npm run build

```

## 💎 Tecnologías

- Gatsby
- Styled Components
- GraphQL
